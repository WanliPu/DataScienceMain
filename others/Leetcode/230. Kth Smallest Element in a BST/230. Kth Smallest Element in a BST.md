# 230. Kth Smallest Element in a BST

## Idea 
* Inorder traversal
* left - > root - > right

## Time 
* O(n)

## Code 
```
public class Solution {
    public int kthSmallest(TreeNode root, int k) {
        Stack<TreeNode> stack = new Stack<TreeNode>();
        while( root != null || !stack.isEmpty() ){
            while( root != null ){
                stack.push(root);
                root = root.left;
            }
            root = stack.pop();
            if(--k == 0 ) break;
            root = root.right;
        }
        return root.val;
    }
}

```
